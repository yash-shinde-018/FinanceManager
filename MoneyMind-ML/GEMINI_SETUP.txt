========================================
GOOGLE GEMINI API SETUP (FREE TIER)
========================================

The chatbot now uses Google Gemini which has a FREE tier!

STEPS TO GET YOUR FREE API KEY:
--------------------------------

1. Go to: https://aistudio.google.com/apikey
   (Alternative: https://makersuite.google.com/app/apikey)

2. Sign in with your Google account

3. Click "Create API Key" or "Get API Key" button

4. Copy the generated API key (starts with "AIza...")

5. Open MoneyMind-ML/.env file

6. Replace "your_gemini_api_key_here" with your actual key:
   GEMINI_API_KEY=AIzaSy...your_actual_key_here

7. Save the file

8. The ML API will auto-reload (or restart it)

DONE! Your chatbot is now ready to use.

TROUBLESHOOTING:
----------------
If you see "chatbot_ready": false:
1. Make sure GEMINI_API_KEY is set in .env
2. Check the API key is valid (starts with "AIza")
3. Restart ML API: python ml_api_v2.py
4. Check logs for any errors
