GROK API SETUP (xAI)
========================================

The chatbot now uses Grok API from xAI (Elon Musk's AI company)!

STEPS TO GET YOUR API KEY:
---------------------------

1. Go to: https://console.x.ai

2. Sign in or create an account

3. Navigate to API Keys section

4. Click "Create API Key" button

5. Copy the generated API key (starts with "xai-...")

6. Open MoneyMind-ML/.env file

7. Replace "your_grok_api_key_here" with your actual key:
   GROK_API_KEY=xai-...your_actual_key_here

8. Save the file

9. Restart the ML API: python ml_api_v2.py

DONE! Your chatbot is now ready to use.

TROUBLESHOOTING:
----------------
If you see "chatbot_ready": false:
1. Make sure GROK_API_KEY is set in .env
2. Check the API key is valid (starts with "xai-")
3. Restart ML API: python ml_api_v2.py
4. Check logs for any errors
5. Verify you have credits in your xAI account
